# siim-isic_melanoma-classification
This repository contains deep/machine learning approaches to identify melanoma in lesion images as part of the [SIIM-ISIC Melanoma Classification Competition](https://www.kaggle.com/c/siim-isic-melanoma-classification). 

# Requirements

1. Python 3.x
2. Tensorflow 2.0
3. Seaborn and matplotlib
4. Pandas
